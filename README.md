# spring-aop
A demo project to learn and implement Spring AOP concepts using practical examples.

Aspect-Oriented Programming (AOP) is an extension of Object-Oriented Programming (OOP) that helps separate common functionalities from the main business logic.

In OOP, the main building block is a class.

In AOP, the main building block is an aspect.

An aspect contains code for tasks that are needed in many places in the application, such as:

Logging
Security
Transaction Management
Exception Handling

These common tasks are called cross-cutting concerns because they are used across multiple classes.

Instead of writing the same code in every class, AOP allows you to write it once in an aspect and automatically apply it wherever needed.
